# Product Roadmap

## Local-first MVP

1. Debate document workspace with structured evidence cards.
2. Reference Bar for hover-to-source interactions.
3. Match room with note template generation and timers.
4. Match history with tags and reflection.
5. AI provider proxy using mock/OpenClaw-compatible/Anthropic adapters.

## Cloud-ready follow-up

1. Workspace and role-based document permissions.
2. Postgres migration.
3. Team evidence library and admin analytics.
4. Real-time collaboration with Yjs once the single-user editor is stable.

## Desktop follow-up

1. Tauri wrapper for the user workspace.
2. Local SQLite storage and optional cloud sync.
3. OS keychain storage if users bring their own API keys.
